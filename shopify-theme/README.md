# LT Shop – Shopify tema

Lietuviška Shopify parduotuvės tema.

## Struktūra

```
shopify-theme/
├── assets/          # CSS ir JS failai
├── config/          # Temos nustatymai
├── layout/          # Pagrindinis layout
├── locales/         # Vertimai (lietuvių kalba)
├── sections/        # Redaguojamos sekcijos
├── snippets/        # Daugkartinio naudojimo komponentai
└── templates/       # Puslapiu šablonai
```

## Diegimas

1. Atsisiųskite `shopify-theme/` aplanką
2. Shopify administravimo skydelyje eikite į **Online Store → Themes**
3. Spustelėkite **Add theme → Upload zip file**
4. Įkelkite archyvą
5. Spustelėkite **Customize** ir pritaikykite pagal poreikius

## Ypatybės

- Pilnai lietuviška sąsaja
- Mobiliesiems pritaikyta (responsive) dizainas
- Šoninis krepšelio staltičius (cart drawer)
- Produktų galerija su miniąturųmis
- Kolekcijų filtravimas ir rūšiavimas
- Greita priežiūra be puslapio perkrovimo
- Online Store 2.0 palaikymas
